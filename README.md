# Atelje Marceta

A web application for showcasing artworks, commissions, and contact information.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Project Structure

- `src/components` – React components (About, Gallery, Contact, Footer, etc.)
- `src/pages` – Application pages (Index, NotFound)
- `src/contexts` – React contexts (e.g., LanguageContext)
- `src/assets/gallery` – Gallery images

## Build for Production

To create a production build:
```sh
npm run build
```
